WCAG Zoom Test Page

This page is designed to test how well the layout, text, and interactive elements respond to browser zooming (e.g., up to 200%). Try using Ctrl + + (or Cmd + + on Mac) to zoom in.

Form Example

Layout Section

This section contains regular content with a responsive layout. As you zoom in, this content should remain readable and stay within the visible browser window without requiring horizontal scrolling.

Tip: WCAG 2.1 Success Criterion 1.4.4 requires support for text scaling up to 200% without loss of content or functionality.
