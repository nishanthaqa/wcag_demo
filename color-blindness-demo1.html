Red
Green
Dark Red

⚠️ Avoid using color alone to convey meaning. Add labels, icons, or patterns.

Good Practice

  • ✅ Use sufficient contrast (WCAG AA: 4.5:1 for normal text).
  • ✅ Use color and shape, icons, or text for differentiation.
  • ✅ Use accessibility tools to simulate color blindness (e.g., Chrome DevTools or Coblis).

Simulating Color Blindness

You can test this page using Chrome's DevTools → Rendering → Emulate vision deficiencies (like Protanopia, Deuteranopia).
